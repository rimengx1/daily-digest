# Daily Digest - 每日精选

> 手动内容管理 + 自动生成网页

---

## 📋 **使用流程**

### **1. 你每天发给我内容**
格式：
```
日期：2026年2月15日

【AI新闻】
1. 标题：OpenAI发布GPT-5
   摘要：简短描述
   链接：https://...

2. 标题：...

【个人思考】
1. 内容：你的思考...
   时间：17:30

2. 内容：...

【好文推荐】
1. 标题：文章名
   来源：网站名
   链接：https://...
```

### **2. 我生成网页**
- 更新数据
- 构建网站
- 部署上线

### **3. 你查看效果**
链接：https://your-site.vercel.app

---

## 🎨 **功能特点**

| 功能 | 说明 |
|------|------|
| 🌍 中英文切换 | 右上角 EN/中 按钮 |
| 📅 往日回顾 | 底部显示历史日期 |
| 📱 响应式设计 | 手机/平板/桌面都适配 |
| ⚡ 静态网站 | 加载速度快 |

## 📝 **内容格式示例**

## 🚀 **部署**

```bash
npm install
npm run build
vercel --prod
```

---

## 📁 **文件说明**

| 文件 | 用途 |
|------|------|
| `index.html` | 页面结构 |
| `style.css` | 样式（Noahpinion黑白风） |
| `app.js` | 交互逻辑 + i18n |
| `content.json` | 每日内容数据 |
| `content-example.json` | 内容格式示例 |
